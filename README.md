Animated Website Interface (HTML & CSS)
About

This project is a simple animated website interface built using only HTML and CSS. It focuses on clean design, smooth animations, and modern UI styling without using JavaScript.

Features

Smooth CSS animations and transitions

Responsive and clean layout

Modern UI design

No JavaScript used

Technologies Used

HTML5

CSS3 (Animations & Transitions)

Usage

Open the index.html file in any modern web browser to view the website.

Purpose

This project is created for learning frontend basics, improving CSS animation skills, and showcasing UI design on GitHub.
